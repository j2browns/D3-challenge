Correlations Discovered Between Health Risks and Age, Income

The above plot allows the user to compare state data for poverty, household income, against the % of the population lacking healthcare and the % suffering from obesity.

The horizontal and vertical axis parameter can be selected by clicking on the desired metrics to be compared. The graph then updates the data and redraws the plot.

Also, if the mouse if moved over an individual point a text box is shown beside the point which shows the 4 data points (%poverty, income, %lacking healthcare, and % with Obesity). Therefore, the user can get the actual values rather then just an estimate from teh graph.
